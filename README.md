# 🏢 Le Comptoir des Artisans - Premium Concept Store Landing Page

A luxurious, fully responsive, bilingual landing page framework built from scratch to modernize the web presence of a landmark local business in Cocody, Abidjan.

## 🎯 Project Motivation & Purpose

Living in the local area, I walked past Rue du Lycée Technique and discovered this beautiful concept store, artisanal boutique, and restaurant. I realized that despite its physical elegance and rich history, its official digital footprint was dead or non-existent, forcing them to rely solely on social media. I engineered this premium platform as a high-utility portfolio project to give this business a professional, multi-page web architecture that matches its luxury real-world brand.

## 🎨 Architectural Design & Layout Features

- **Luxury Brand Theming**: Structured using a premium color palette consisting of rich wood dark charcoal (`#1a1512`) and metallic gold outlines (`#d4af37`) to match the Afro-European aesthetic of the venue.
- **Bilingual Front-Door Entry**: Created synchronized master homepages in both English (`index.html`) and French (`index-fr.html`) connected via instant header language navigation toggles.
- **Two-Axis Layout Physics**: Powered by a robust Flexbox positioning system using `justify-content: space-between` and `align-items: center` to suspend components flawlessly.
- **Adaptive Mobile Grid Shields**: Engineered with `flex-wrap: wrap` safety valves and flexible width sizing parameters (`width: 100%; max-width: 500px;`) to guarantee pixel-perfect presentation on modern smartphone screens.

## 🛠️ Built With

- **HTML5**: Highly organized semantic layout containers.
- **CSS3**: Modern Flexbox multi-axis positioning rules and unified font scales.
- **Git & GitHub**: Distributed version control pipelines over secure Port 443 SSH links.
